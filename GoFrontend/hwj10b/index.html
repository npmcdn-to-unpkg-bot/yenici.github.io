Скопировать себе в script.js data.json, сделать из него обычный javascript-массив (распарсить). С помощью lodash получить несколько массивов:
  1. Массив скиллов (поле skills) всех людей, не должно быть повторяющихся скиллов, так же они должны быть отсортированы по алфавиту; 
                var data = getData();
            var arr = _.sortBy(
              _.uniq(
                _.flatten(
                  _.map(data, 'skills')
                )
              )
            );

  2. Массив имен (поле name) людей, отсортированных в зависимости от количества их друзей (friends); 
                var data = getData();
            var arr = _.map(
              _.sortBy(data, ['friends', 'name']),
              'name'
            );

  3. Массив всех друзей всех пользователей, не должно быть повторяющихся людей. 
                var data = getData();
            var arr = _.sortBy(
              _.uniq(
                _.map(
                  _.flatten(
                    _.map(data, 'friends')
                  ),
                  'name'
                )
              )
            );
