# Урок 17-18.1

Домашнее задание:

1. Установить [node.js](https://nodejs.org/en/)
2. Установить [grunt.js](http://gruntjs.com/)
3. Настроить сборку проекта простого. Должны выполняться задачи по конкатенации и минификации ваших `js` и `css` файлов
4.
